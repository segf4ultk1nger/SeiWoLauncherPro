## 🌟 愿景

**SeiWoLauncherPro** 不仅仅是一个桌面助手。它是对臃肿、陈旧的教学环境 UI 的一次“优雅叛逆”。我们追求极致的 **UX 体验**，将现代设计语言引入封闭的希沃生态，为师生提供一个美观、流畅且高度可定制的数字化工作空间。

## ✨ 核心特性

### 🎨 工业级美学设计

* **iOS 级视觉体验**：原生实现 **R2 曲率连续圆角** 与**实时高斯模糊**，打破 WPF 传统的“Windows 7 遗风”。
* **自研组件库**：拒绝套用通用的 UI 框架，绝大多数控件均为底层自研，确保动效丝滑，每一帧都经得起推敲。

### 📦 极致的图标与空间管理

* **高级收纳**：支持多页布局、文件夹分类及自定义长按菜单。
* **全生命周期钩子**：内置 `prerun` / `postrun` 脚本支持，为专业用户提供无限可能。
* **高度自定义**：图标、名称、快速启动适配，你的桌面你做主。

### 🛠️ 强大的扩展生态

* **小组件系统**：内置 U 盘管理、PPT 状态、媒体控制等核心组件。
* **信息一览**：在轻量级环境下完美替代 ClassIsland，关键信息抬眼即见。
* **插件化架构**：支持使用 C# 编写第三方插件，逻辑与表现完全解耦。

### 🚀 深度集成希沃生态

* **UDI 协议适配**：直接调用希沃五代及以上硬件的安卓通道，跨系统调节参数不再是梦。
* **全家桶联动**：深度对接希沃桌面，并预留了 **SeiWoDesktopPro** 的未来接口。
